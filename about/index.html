Giorgio Dell'Immagine

Computer science student, math and logic enthusiast, GNU/linux user and clarinet player.

Hi, I’m Giorgio Dell’Immagine. I am currently a computer science student at University of Pisa. My main interests are: logic, abstract algebra, cathegory theory, formal systems, functional programming, meta programming, programming language theory and classical music.

I’ve worked on a formal verification tools using Hoare logic, tools for pulsar hunting and signal elaboration, and a lot of parsers, interpreters and simple compilers.

The languages I use daily: C, python, Haskell, lisp, Latex, and many more!
